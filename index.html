Progressive Enhancement

What?

A development process that places the content at the core of the digital experience.

It uses web technologies in a layered fashion allowing any user to access the basic content and functionality of a web page, using any browser or Internet connection, while also providing an enhanced version of the page to those with more advanced browser software or greater bandwidth.

Why?

Digital content created using this process is by nature more accessible, as the strategy demands that basic content always be available, not obstructed by commonly unsupported or easily disabled scripting.

Additionally, the sparse markup principle makes it easier for tools that read content aloud to find that content and the basic content is always accessible to search engines.
